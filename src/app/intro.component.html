Welcome to the AppStream MetaInfo Creator!

This simple tool lets you create AppStream MetaInfo files by answering a simple set of questions.

Please select the type of software that you want to generate metadata for:

GUI Application

GUI applications are application which have a graphical user interface and are commonly used with mouse and keyboard. They also ship a Freedesktop desktop-entry file to be visible in application menus.

Create GUI Application Metadata

Console Application

Console application are any application that have a command-line or text-based interface and are designed to be used by a human user in a console. Their binaries need to be present in PATH.

Create Console Application Metadata

Addon

Addons are software components that are not standalone applications but intended to be used with another software component, usually extending the functionality of the application they extend.

Create Addon Metadata

Service

Services are background processes ("daemons") providing a general system ability. They can for example be webservers, mailservers, webapps (basically anything that is started by the init system, usually systemd).

Create Service Metadata

More Software Types

AppStream provides support for describing a lot more software components than just the ones listed here, such as fonts, firmware, drivers, localization or whole operating systems.

Check out the specification to learn how to write metadata for these types of software and data!
